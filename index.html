Current Command

git init creates a new Git repository. It creates a hidden .git directory that contains all the necessary files and directories for Git to track changes.

git init [directory-name]

Legend

Commit: A snapshot of your changes at a specific point in time. Each commit has a unique ID and contains the changes made.

Branch: A parallel line of development. Branches allow you to work on different features independently.

HEAD: A pointer to the current commit you're working on. It moves automatically as you make new commits or switch branches.

Remote: A copy of your repository hosted on a server (like GitHub). Used for collaboration and backup.
